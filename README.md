# Proyectos C#

Este repositorio contiene varios proyectos desarrollados en C#.

## Proyecto 1: **CineApp Windows Form**

**Descripción**:  
CineApp es una aplicación de escritorio basada en C# para la compra de entradas, comida y bebidas. Utiliza la interfaz de línea de comandos (CLI) para calcular el total de la compra.

**Características**:
- Compra de entradas para películas.
- Selección de comida y bebidas.
- Cálculo automático del total.

## Proyecto 2: **ProyectoCine.net**

**Descripción**:  
ProyectoCine.net es una versión más avanzada de CineApp, que incluye una interfaz gráfica. Además, distingue las películas y proporciona una mejor experiencia de usuario.

**Características**:
- Interfaz gráfica de usuario.
- Selección de película, comida y bebidas.
- Cálculo del total.
- Mejora en la usabilidad.

## Proyecto 3: **ASP.NET CRUD**

**Descripción**:  
Este proyecto es una aplicación web basada en ASP.NET que implementa un CRUD (Crear, Leer, Actualizar, Eliminar) completo. Utiliza Entity Framework para interactuar con la base de datos.

**Características**:
- Interfaz web.
- Implementación de un CRUD completo.
- Uso de Entity Framework para gestión de datos.
- Seguridad y control de acceso.

## Proyecto 4: **Pizzería Web**

**Descripción**:  
El proyecto Pizzería es una aplicación web completa para la venta de pizzas. Incluye un sistema de pago propio, además de tener funcionalidades de administración para gestionar productos y usuarios.

**Características**:
- Página web de pizzería completa.
- Sistema de pago propio.
- Funciones de administración para gestionar productos, usuarios y pedidos.
- Uso de Entity Framework y SQL Server.
- Diferenciación entre usuarios y administradores.
